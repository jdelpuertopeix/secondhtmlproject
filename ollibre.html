Llistes ordenades (ol)

Les llistes ordenades serveixen per a definir una seqüència ordenada de passos. Un bon exemple són les receptes de cuina

Recepta de truita de patates

  1. Pelar les patates
  2. Escalfar la paella amb l'oli d'oliva
  3. Tallar les patates ben fines a rondanxes
  4. Quan l'oli està calent, fregir les patates
  5. Batre els ous i afegir-hi all i julivert
  6. Afegir els ous amb les patates
  7. Donar-li la volta
  8. Ja podem servir!

Aquests són els passos per a fer una truita de patates deliciosa.
